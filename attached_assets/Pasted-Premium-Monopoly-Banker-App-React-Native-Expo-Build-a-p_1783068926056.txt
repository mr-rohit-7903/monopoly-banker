Premium Monopoly Banker App (React Native Expo)

Build a production-quality React Native application using Expo that works as a complete Monopoly Banking Companion for one shared mobile device acting as the game's banker.

The application must function 100% offline, be fast, beautiful, and feel like an official premium board game companion app.

Tech Stack

Use:

React Native
Expo (latest stable version)
TypeScript
Expo Router (file-based routing)
React Query (if needed for async local operations)
Zustand or Redux Toolkit for state management (prefer Zustand)
SQLite (expo-sqlite) or Realm for local persistence
React Native Paper or NativeWind (Material 3 styling preferred)
React Hook Form
Zod for validation
React Native Reanimated (only where beneficial)
FlashList for high-performance lists
